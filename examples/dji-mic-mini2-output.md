# DJI Mic Mini 2 无线降噪麦克风（¥350）—— 预期分析输出

这是 cost-anatomist skill 的第一个验收案例，也是测试时的预期效果参考。

## 测试输入

```
我刚花了350块买了个大疆 Mic Mini 2 无线降噪麦克风，帮我拆一下成本
```

## 预期输出要点

### 品类识别
- 自动识别为"消费电子-音频设备"
- 匹配"消费电子"成本基线

### 应搜索的关键方向
1. DJI 大疆创新的毛利率/净利率（官方未上市，但行业估算毛利率 40-50%）
2. 歌尔股份为 DJI 麦克风代工的财报数据
3. 同类产品（罗德 Wireless GO II）的拆机/BOM 分析
4. DJI Mic Mini 2 的核心元器件（AKM 音频芯片、MEMS 麦克风单元等）的公开报价
5. 淘宝/拼多多渠道的经销商利润空间

### 预期成本拆解（大致估计）

| 维度 | 占零售价 | 依据 |
|------|---------|------|
| 物料成本(BOM) | ~20-25% (¥70-88) | 小体积音频设备 BOM 占比偏低，AKM 芯片+DSP+电池+外壳 |
| 制造/代工 | ~5-8% (¥18-28) | 歌尔股份代工净利 3-5%，总代工成本含人工 |
| 研发摊销 | ~8-12% (¥28-42) | DJI 音频算法（降噪/增益控制）+ 无线协议研发 |
| 营销推广 | ~8-12% (¥28-42) | B站/小红书数码博主投放 + 发布会/展会 |
| 渠道加价 | ~18-25% (¥63-88) | 经销商 + 电商平台(淘宝/京东抽成 5-8%)，DJI 官网直营会低 |
| 物流仓储 | ~3-5% (¥11-18) | 小件物流成本低 |
| 品牌溢价 | ~12-20% (¥42-70) | DJI 品牌在无人机/影像领域的口碑溢价 |
| 税费 | ~10-13% (¥35-46) | 增值税 13% |
| 售后/保修 | ~2-3% (¥7-11) | 一年质保+退换 |
| 企业净利 | ~8-12% (¥28-42) | 估算 DJI 此产品净利 |

### 竞品对比
- 罗德 Wireless GO II：约 ¥1,500-2,000，功能类似但品牌溢价更高
- 猛玛 Lark M1：约 ¥200-300，功能基础但性价比更高
- 对比结论：DJI Mic Mini 2 在 ¥350 价位段性价比相当高

### 预期"值不值"判定
- 判定：**值回票价**（甚至是"合理偏便宜"）
- 理由：¥350 买一个有 DJI 降噪算法+无线稳定传输的麦克风，物料+研发成本已经吃掉了一半以上，品牌溢价也在合理范围。相比之下竞品在这个价位段基本没有同等降噪能力。
