# 成本分析框架

定义不同品类需要分析的维度组合、特殊维度、以及各维度的搜索策略。

---

## 通用消费品分析维度（10+1）

按从原材料到消费者手中的链条顺序排列：

```
物料(BOM) → 制造/代工 → 研发摊销 → 物流仓储 → 营销推广 → 渠道加价 → 品牌溢价 → 税费 → 售后预留 → 企业利润
                                                                                        ↓
                                                                                   （消费者到手）
```

每个维度的分析应包含：
1. **估算金额**（¥）
2. **占零售价百分比**
3. **估算依据**（硬数据引用或行业基线参考）
4. **可信度评级**（⭐1-5）
5. **通俗解释**（150-300 字，讲清楚"这部分钱给了谁、为什么给这么多"）

---

## 数码电子产品附加维度

在通用维度基础上增加：

- **芯片/核心元件成本**：从 BOM 中单独拎出来讲，因为这是电子产品最核心的成本项
- **专利许可费**：高通专利费（手机售价 3-5%）、Dolby/HDR 等标准授权
- **代工厂利润细分**：富士康/立讯/歌尔等代工厂模式，通常净利 2-5%
- **关税/贸易政策影响**：中美关税、产地（越南/印度/中国）对价格影响
- **首发溢价 vs 降价周期**：电子产品上市后 6-12 个月的降价规律

**电子产品搜索策略**：
- 优先搜索 iFixit 拆机报告、Zealer/爱否科技等科技媒体的 BOM 分析
- 搜索代工厂财报（如歌尔股份为 DJI 麦克风代工，其毛利率公开）
- 搜索芯片原厂报价（Qualcomm/TI/AK 芯片的公开千颗批发价）

## 化妆品/护肤品附加维度

- **活性成分成本**：从 BOM 中拆出"有效成分"vs"基质/溶剂"的成本
- **包装的隐性成本**：奢侈级玻璃瓶、磁吸盖、镀金件的额外成本
- **明星代言费摊销**：某代言人年费 ÷ 产品年销量 = 每件产品的代言成本
- **渠道的恐怖抽成**：百货专柜 25-35% vs 免税店 10-20% vs 电商 5-15%

**化妆品搜索策略**：
- 搜索品牌母公司（L'Oréal/Estée Lauder/资生堂）财报的毛利率和营销费用率
- 搜索代工厂（科丝美诗/诺斯贝尔）的 ODM/OEM 报价
- 搜索"化妆品成分成本""大牌成分成本占比"类科普文章

## 服务类产品分析维度

服务与实体商品的结构不同，分析维度调整为：

```
人工成本 → 场地/设备 → 资质/合规 → 营销获客 → 管理运营 → 税费 → 企业利润
```

**服务类特殊考量**：
- **单位时间定价法**：服务报价 ÷ 预计工时 = 时薪，与行业基准时薪对比
- **设备摊销法**：昂贵仪器（如医美设备）按使用次数摊销到每次服务中
- **获客成本**：服务类获客成本通常很高（教育/医美/保险 CAC 可达客单价 30-50%）
- **信息不对称溢价**：专业服务的"知识溢价"部分（律师/咨询/培训）

## 特殊产品类型的额外分析维度

### 限量版/联名款
- 联名授权费（如迪士尼联名、艺术家联名）
- 限量制造的额外开模/小批量生产成本
- **稀缺性溢价**：限量编号带来的二级市场溢价

### 盲盒/概率型产品
- 单盒期望成本 vs 标价
- "集齐全套"的期望总成本
- 隐藏款的真实概率和二级市场价格
- 商家通过概率设计赚取的概率溢价

### 订阅制/会员制
- 用户终身价值(LTV) vs 获客成本(CAC)
- 首单补贴/拉新成本
- 续费率对定价的影响

### 进口商品
- 关税阶梯（不同品类关税差异大）
- 汇率影响
- 跨境物流成本
- 中国区定价 vs 原产国定价的"价格歧视"分析

## 来源可信度评级标准

| 星级 | 含义 | 示例来源 |
|------|------|---------|
| ⭐⭐⭐⭐⭐ | 官方公开精确数据 | 上市公司财报明细、品牌官方成本披露 |
| ⭐⭐⭐⭐ | 可信第三方精确数据 | iFixit 拆机 BOM 清单、知名分析师报告 |
| ⭐⭐⭐ | 行业报告/专家估算 | 券商研报、行业白皮书、头部媒体分析 |
| ⭐⭐ | 行业经验区间 | 品类通用成本结构、从业者口述 |
| ⭐ | 纯推测/类比 | "根据类似产品推测""行业惯例可能是" |

## 竞品对比维度

选择 2-3 款竞品时优先考虑：
1. 同价位的直接竞品
2. 同品牌的上下级产品（如 DJI Mic 系列的其他型号）
3. 品类标杆产品（作为参照基准）

对比字段：零售价、已知成本结构、核心参数、性价比评分（功能/价格比）
